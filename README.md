## **注意事項**

💡

1. 當前畫面停留 1 分鐘後，時間到將清除所有交易流程紀錄。                                                    
2. 當網頁關閉並重新開啟，若超過 1 分鐘則清除網頁資訊，否則繼續交易。
3. 輸入交易數量時，若超過持有數量，則顯示 alert【餘額不足】。
